OCx24 Dataset

Dashboard to visualize the experimental dataset in this work: https://ai.meta.com/blog/open-catalyst-simulations-experiments/

This dataset contains electrochemical performance data for metal alloy catalysts tested for CO₂ reduction and hydrogen evolution reactions. The samples were synthesized using two different methods:

  • Chemical Reduction: Produces powder catalysts
  • Spark Ablation: Produces porous film catalysts

All catalysts were applied to Gas Diffusion Electrodes (GDEs) and tested in Membrane Electrode Assemblies (MEAs) in neutral conditions using 0.1M KHCO₃ electrolyte. The electrochemical testing was performed in a two-electrode configuration with direct voltage measurement. Gas products were analyzed using Gas Chromatography (GC).

Each synthesis method includes three duplicate samples for statistical reliability. All samples contain X-ray Fluorescence (XRF) composition data, and most samples include X-ray Diffraction (XRD) structural characterization data.
